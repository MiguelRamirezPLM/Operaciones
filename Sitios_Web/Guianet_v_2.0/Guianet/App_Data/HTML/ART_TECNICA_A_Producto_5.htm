Cámara colapsable - VentChamber

DESCRIPCIÓN:

Cámara colapsable diseñada para optimizar el depósito del aerosol (IDM) en pacientes sometidos a ventilación mecánica.

CARACTERÍSTICAS:

• La cámara se coloca en la rama inspiratoria del circuito y puede colapsarse cuando no está en uso, manteniendo el circuito cerrado.

• Se puede usar con todos los IDM, incluyendo los que tienen contador de dosis.
